# Logo Assets

Place your logo files in this folder (`web/public/logo/`).

## Required Files:

1. **logo.svg** - Main logo (SVG format recommended)
   - Used as the app icon and throughout the application
   - Should be a square or circular design
   - Recommended size: 512x512px viewBox

## How to Add Your Logo:

1. Create or export your logo as an SVG file
2. Name it `logo.svg`
3. Place it in this folder: `web/public/logo/logo.svg`
4. The app will automatically use it

## Logo Usage in the App:

The logo is used in:
- Browser tab favicon
- App header/navigation
- Login page
- Mobile app icon (when installed as PWA)

## Tips for PWA Icons:

- Use a simple, recognizable design
- Ensure it looks good at small sizes (favicon)
- Test on both light and dark backgrounds
- Keep the SVG optimized for web

## Alternative: PNG Icons

If you prefer PNG, you can also place these files in `web/public/icons/`:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

These are used for the PWA installable app icons.

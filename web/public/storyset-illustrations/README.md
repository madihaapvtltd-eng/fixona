# Storyset Illustrations

This folder contains vendor/illustration images for the MadManRep CMMS application.

## Image Sources

Images sourced from [Storyset](https://storyset.com) - Free customizable illustrations for your projects.

## Usage

Add illustration files here for use in:
- Login page backgrounds
- Empty state illustrations
- Error page graphics
- Dashboard decorative elements
- Onboarding slides

## Naming Convention

- Use descriptive names: `maintenance-team.svg`, `empty-workorders.svg`
- Use kebab-case for multi-word names
- Preferred format: SVG (scalable) or PNG (transparent background)
